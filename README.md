## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

1. Code editor.
2. Git.
3. [Ruby](https://www.ruby-lang.org/en/)

### Setup

In order to clone this repository to your desired folder, run the commands below on your console:

```sh
    cd my_folder
    git clone https://github.com/Luiscarlosvd/test-driven-development.git
```

### Install

Install this project with:

```sh
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby solver.rb
```

### Test

To run the tests in the project, execute the following command:

```sh
  gem install rspec
  rspec spec
```
